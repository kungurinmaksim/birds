# 🐦 ПТИЦЫ!!!

**Аркадная игра-кликер на Python с Tkinter**, где вы управляете летающим персонажем, отбивающимся от стен, и зарабатываете очки, кликая по нему мышкой.

## 🎮 Об игре

Игра представляет собой аркадный симулятор, где персонаж (птичка или другой выбранный вами герой) летает по экрану, отскакивая от стен. Ваша задача — кликать по нему мышкой, чтобы замедлять его. Каждый раз, когда персонаж касается нижней стены, его скорость увеличивается. Игра заканчивается, если скорость становится слишком высокой (≥13) или падает до нуля.

### 🌟 Особенности

- 🖱️ **Кликерная механика** — скорость персонажа меняется при кликах по нему
- 🎨 **Выбор персонажей** — 4 разных героя на выбор
- 📈 **Система очков** — скорость увеличивается при касании низа и уменьшается при кликах
- 🔴 **Индикация опасности** — скорость ≥8 отображается красным цветом
- 🎯 **Увлекательный геймплей** — нужно балансировать скорость, чтобы не проиграть
- 🎨 **Анимация персонажей** — каждый герой имеет анимацию из 4 кадров

## 🕹️ Управление

| Действие | Управление |
|----------|------------|
| **Клик по персонажу** | Уменьшение скорости на 1 |
| **Клик по пункту меню** | Навигация в главном меню |

### Главное меню

| Пункт меню | Действие |
|------------|----------|
| **Новая игра** | Начать игру |
| **Персонажи** | Открыть меню выбора персонажа |
| **Выход** | Закрыть игру |

### Меню выбора персонажей

| Пункт меню | Персонаж |
|------------|----------|
| Злая птичка | Агрессивная птица |
| Птичка | Обычная птица |
| Добрая птичка | Милая птица |
| Муха | Насекомое |
| Главное меню | Вернуться в главное меню |

## 📸 Как играть

1. **Запустите игру**
2. В главном меню выберите:
   - `Новая игра` — начать игру со стандартными настройками
   - `Персонажи` — выбрать внешний вид героя
   - `Выход` — закрыть игру
3. **Игровой процесс**:
   - Персонаж летает по экрану, отскакивая от стен
   - Каждое касание нижней стены увеличивает скорость на 1
   - Клик по персонажу уменьшает скорость на 1 и увеличивает счёт
   - Скорость отображается в центре экрана
   - Счётчик очков показывает количество успешных кликов
4. **Условия поражения**:
   - Скорость ≥ 13 — игра заканчивается
   - Скорость ≤ 0 — игра заканчивается

## 🔧 Технические детали

### Параметры игры
- Размер игрового поля: `400×300` пикселей
- Размер персонажа: `100×100` пикселей
- Частота анимации: `50 мс` (≈20 FPS)
- Частота обновления физики: `30 мс` (≈33 FPS)
- Начальная скорость: `5`
- Направление движения: диагональное (по диагонали вправо-вниз)

### Механики
- **Физика отскока**: При столкновении со стенами направление меняется на противоположное
- **Скорость**: Влияет на расстояние перемещения за каждый кадр
- **Система очков**: `i` увеличивается при кликах по персонажу
- **Индикатор опасности**: При скорости ≥8 текст становится красным

### Алгоритм движения
```python
Каждый кадр:
    1. Проверить столкновение со стенами
    2. Проверить клик по персонажу
    3. Обновить позицию: x += vx * speed, y += vy * speed
```

## 🎯 Система очков и скорости

| Действие | Влияние на скорость | Влияние на счёт |
|----------|---------------------|-----------------|
| Касание нижней стены | +1 | — |
| Клик по персонажу | -1 | +1 |

### Критические значения скорости

| Скорость | Состояние |
|----------|-----------|
| 0 | Поражение (слишком медленно) |
| 1-4 | Опасная зона |
| 5-7 | Нормальная зона |
| 8-12 | Красная зона (опасность) |
| 13+ | Поражение (слишком быстро) |

### Возможные проблемы

**Ошибка `tkinter not found`**:
- Linux: `sudo apt-get install python3-tk`
- Windows/Mac: переустановите Python

**Ошибка загрузки изображений**:
- Убедитесь, что файлы названы по шаблону `e_{i}s_{s}.png`
- Проверьте наличие всех 4 кадров анимации для каждого персонажа

## 🎨 Интерфейс

### Элементы интерфейса
- **Синяя линия внизу** — обозначает опасную зону (касание ускоряет персонажа)
- **Счётчик скорости** — отображается в центре экрана (красным при скорости ≥8)
- **Счётчик очков** — отображается выше счётчика скорости
- **Фон** — загружается из файла `images.png`

### Меню
- Текст отображается по центру экрана
- Для выбора пункта достаточно кликнуть по тексту
- Меню персонажей позволяет выбрать скин до начала игры

## 🐛 Известные особенности

- При клике по персонажу он продолжает движение (не останавливается)
- Персонаж может "застревать" в стенах при очень высокой скорости (редко)
- При скорости 0 игра завершается, но персонаж продолжает лететь (баг)
